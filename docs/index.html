Quick Start

Nemosyne automatically generates VR scenes from your data. Just provide the data—it builds the optimal visualization.

Installation

# Via npm
npm install nemosyne-vr

# Via CDN src="https://unpkg.com/nemosyne-vr@latest/dist/nemosyne.min.js"></script>

Basic Usage

// Initialize
const engine = new DataNativeEngine();

// Your data
const data = {
  nodes: [{ id: 1, value: 10 }, { id: 2, value: 20 }],
  links: [{ source: 1, target: 2 }]
};

// Auto-render
engine.loadData(data).then(() => engine.render('#scene'));

API Reference

DataNativeEngine

constructor(options)

Creates a new Nemosyne instance.

options.autoDetect
Auto-select visualization (default: true)
options.enableGestures
Enable hand tracking (default: true)
options.performanceMode
'quality', 'balanced', or 'performance'

loadData(data): Promise

Load data and auto-detect topology.

render(selector): void

Render visualization to DOM element or A-Frame scene.

TemporalScrubber

play(): void

Start time animation.

scrubTo(timestamp): object

Jump to specific time point.

MemPalaceVRConnector

connect(): Promise

Connect to MemPalace API.

findMemoriesNearPosition(position, radius): Array

Spatial query—find memories within radius.

Examples

Animated Memory Explorer

Walk through your MemPalace with time as a dimension:

const connector = new MemPalaceVRConnector();
const scrubber = new TemporalScrubber();

await connector.connect();
await scrubber.loadFromMemPalace(connector);

scrubber.play(); // Watch memories evolve
