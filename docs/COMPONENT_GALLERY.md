# Component Gallery

A comprehensive showcase of Nemosyne's artefact types and visual components.

## Overview

This document provides a visual reference for all available 3D visualization components in the Nemosyne framework.

## Artefact Types

### Basic Geometries
- **NemosyneBar** - 3D bar chart with height encoding
- **NemosyneScatter** - Point clouds with position/size/color encoding
- **NemosyneLine** - Connected line segments

### Topology Visualizations
- **NemosyneGraph** - Force-directed network visualization
- **NemosyneTree** - Hierarchical tree layouts
- **NemosyneTimeline** - Temporal data with linear or spiral layouts

### Specialized
- **NemosyneGlobe** - Geographic data on 3D globe
- **NemosyneMatrix** - Correlation matrices as 3D surfaces
