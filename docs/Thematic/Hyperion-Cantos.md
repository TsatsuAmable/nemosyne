# Hyperion Cantos Thematic Artefacts

Nemosyne extends into science fiction with artefacts inspired by Dan Simmons' Hyperion Cantos.

---

## The Datumplane

### Concept

The Datumplane is a virtual-reality construct where data takes on form and substance—the "datasphere" equivalent from the Hyperion universe.

"*The datumplane had opened around him like an origami flower, each petal a different slice of reality...*"

### Key Concepts

| Concept | Description | Implementation |
|---------|-------------|----------------|
| **Datasphere** | Virtual space for information | VR environment |
| **Farcasters** | Instantaneous travel portals | Portals between scenes |
| **AI Cores** | Artificial intelligences | Animated entities |
| **TechnoCore** | Vast AI collective | Swarm intelligence |
| **The Shrike** | Time-travelling entity | Temporally dynamic |
| **Time Tombs** | Reverse-time structures | Countdown visualizations |
| **Treeship** | Organic spacecraft | Organic branching structures |
| **Ousters** | Gene-modified humans | Avatar representations |

## Artefact Catalogue

### Shrike - The Lord of Pain

**Description:**
A metallic, four-armed entity with thorns and blades. Moves backwards through time, appearing before it arrives.

**Geometry:**
- Central body: Icosahedron with spikes
- Four arms: Extending tetrahedrons
- Blades: Intersecting transparent planes
- Aura: Red emissive glow

**Behaviours:**
- **Temporal Paradox**: Appears at destination before departure point
- **Blade Field**: Surrounding geometric shards rotate
- **Time Freeze**: Stops local time for debugging
- **Prediction**: Shows future position as ghost

**Data Binding:**
```javascript
{
  id: 'shrike-entity',
  type: 'temporal-sentinel',
  data: {
    temporalPosition: 'reverse',
    velocity: { x, y, z },
    predictors: [futureState1, futureState2],
    menace: 0.95
  }
}
```

**Usage Scenario:**
Critical system monitoring. The Shrike appears *before* system failures, giving time to react.

### Time Tomb

**Description:**
Reverse-time structures that count down to their opening. What exists now will be buried; what is buried will emerge.

**Geometry:**
- Exterior: Crystal formation (opalescent)
- Entropy gradient: Order → Chaos
- Clock: Runs backwards
- Portal: Elliptical opening

**Behaviours:**
- **Reverse Entropy**: Things become more ordered as time passes
- **Memory Release**: Past events play in reverse
- **Temporal Lock**: Cannot be opened until countdown complete
- **Paradox**: Opening causes events that lock the tomb

**Data Binding:**
```javascript
{
  tombId: 'tomb-of-the-wind',
  countdown: 86400000, // Until opening
  contents: ['prediction', 'warning', 'past-event'],
  entropy: 'increasing-order',
  locked: true
}
```

**Usage Scenario:**
Deadlines, launches, scheduled events. The tomb opens precisely when scheduled, not before.

### Farcasters - Portal Network

**Description:**
Instantaneous travel portals that connect distant locations in the datasphere.

**Geometry:**
- Portal frame: Toroidal structure
- Event horizon: Transparent disc
- Particle stream: Flowing through center
- Destination coordinates: Floating runes

**Behaviours:**
- **Instantaneous**: Teleportation (scene transition)
- **Connection Status**: Linked/Unlinked
- **Data Flow**: Visual of information transfer
- **Cooldown**: Recharge animation

**Data Binding:**
```javascript
{
  portalId: 'farcaster-alpha',
  destination: 'scene-id-123',
  connection: 'established',
  bandwidth: 'unlimited',
  latency: 0,
  security: 'encrypted',
  status: 'active'
}
```

**Usage Scenario:**
Quick navigation between related data spaces. Jump from overview → detail → timeline instantly.

### Memory Crystal (Mnemosyne)

**Description:**
Crystalline storage of perfect memory. Accessing one relives the experience perfectly.

**Geometry:**
- Crystal: Octahedron or dodecahedron
- Internal structure: Fractal patterns
- Refractive index: Varies with data density
- Stored pattern: Visible within facets

**Behaviours:**
- **Playback**: Rotate to view different angles (perspectives)
- **Refraction**: Data visible through different facets
- **Memory Projection**: Cast stored image as hologram
- **Crystal Growth**: Expand as more data added

**Data Binding:**
```javascript
{
  crystalId: 'memory-42',
  timestamp: '2026-04-09T00:00:00Z',
  data: { /* Any structure */ },
  emotions: ['awe', 'curiosity'],
  clarity: 1.0,
  accessibility: 'instant'
}
```

**Usage Scenario:**
Snapshot visualization. Save current state exactly and revisit anytime.

### Templar Tree

**Description:**
Sacred structure of the Templars. Branching, crystalline, pulsing with energy.

**Geometry:**
- Trunk: Central octagonal pillar
- Branches: Recursive branching fractals
- Leaves: Crystal facets
- Core: Pulsing energy source

**Behaviours:**
- **Seasonal Change**: Colours shift (data age)
- **Wind Response**: Slight sway based on activity
- **Blossom**: New leaves appear (new data)
- **Entanglement**: Connected to other trees

**Data Binding:**
```javascript
{
  treeId: 'templar-shrine-7',
  age: 120, // In data years
  branches: 64,
  leaves: 4096,
  energyFlow: 'upward',
  seasons: ['spring-data', 'summer-active', 'autumn-archive', 'winter-quiescent'],
  connectedTo: ['tree-1', 'tree-9']
}
```

**Usage Scenario:**
Hierarchical data visualization. Root = source, branches = categories, leaves = individual records.

## Implementation

### Quick Start

```javascript
// Import Hyperion Cantos themed artefacts
import { Shrike, TimeTomb, Farcasters, MemoryCrystal } from 'nemosyne/thematic/hyperion';

// Create Shrike
const shrike = new Shrike({
  container: '#scene',
  temporalMode: 'reverse',
  menace: 0.9
});

// Create Time Tomb
const tomb = new TimeTomb({
  countdown: 60 * 60 * 1000, // 1 hour
  contents: ['prediction', 'warning']
});

// Create Farcasters
const portal = new Farcasters({
  destination: 'scene-2',
  connection: 'secure'
});
```

### Custom Thematic Pack

```javascript
// Define custom Hyperion artefact
class MyCustomArtefact extends Nemosyne.ThematicArtefact {
  constructor(options) {
    super(options);
    this.theme = 'hyperion-cantos';
    this.visualStyle = 'sci-fi-opera';
  }
  
  createGeometry() {
    // Custom Three.js geometry
    return new THREE.CustomGeometry(...);
  }
}

// Register
Nemosyne.registerThematic('hyperion', 'custom', MyCustomArtefact);
```
