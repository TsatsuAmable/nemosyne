# Nemosyne Wiki

> ⚠️ **RESEARCH PREVIEW**: This is an experimental framework for investigating immersive data visualization. Many features are implemented but untested. [See Research Agenda](../RESEARCH_AGENDA.md)

Welcome to the Nemosyne Research Framework Wiki!

## 🔬 Current Status

Nemosyne is a research project, not a production framework. We're investigating whether 3D VR visualizations improve data comprehension compared to 2D.

| Component | Implementation | Validation |
|-----------|---------------|------------|
| Core Rendering | ✅ Working | ⚠️ Needs UX study |
| Layout Algorithms | ✅ 7 implemented | ❌ Not started |
| Topology Detection | 🚧 Partial | ❌ Not started |
| Performance | 🚧 Unbenchmarked | ❌ Not started |
