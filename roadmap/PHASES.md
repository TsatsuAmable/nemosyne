# Nemosyne Development Roadmap

## Phase 0: Foundation (Weeks 1-2)
**Goal:** Research and theory development

- [ ] Research data topology categories (graphs, fields, time-series, hierarchies, networks)
- [ ] Study VR interaction patterns and affordances
- [ ] Define artefact taxonomy (what types of VR objects work for what data?)
- [ ] Document findings in `/research`

## Phase 1: Specification (Weeks 3-4)
**Goal:** Define the framework's architecture and APIs

- [ ] Artefact specification format (schema for defining artefacts)
- [ ] Behaviour system design (how artefacts respond to events)
- [ ] Transform pipeline (data → 3D representation)
- [ ] Core framework architecture (A-Frame integration, D3.js bindings)
- [ ] Write specification docs

## Phase 2: Core Framework (Weeks 5-8)
**Goal:** Build the foundational library

- [ ] Base Artefact class/component
- [ ] Transform engine
- [ ] Behaviour system
- [ ] Event handling
- [ ] Initial set of primitive artefacts (crystal, node, bar, sphere, etc.)
- [ ] Unit tests

## Phase 3: Hello World + Examples (Weeks 9-10)
**Goal:** Demonstrate the framework's capabilities

- [ ] "Hello World" — simplest possible VR data viz
- [ ] Example 1: Network graph (social connections, dependencies)
- [ ] Example 2: Time-series (stock prices, sensor data over time)
- [ ] Example 3: Geospatial (map data, location clusters)
- [ ] Example 4: Hierarchical (file system, org charts)

## Phase 4: Documentation + Community (Week 11+)
**Goal:** Make it usable by others

- [ ] Comprehensive README
- [ ] API documentation
- [ ] Tutorial: "Building Your First Artefact"
- [ ] Wiki/GitHub Pages site
- [ ] Contribution guidelines

## Phase 5: Artefact Library Expansion (Ongoing)
**Goal:** Build a rich ecosystem of VR data components

- [ ] Advanced artefacts (chord diagrams, Sankey flows, heatmaps)
- [ ] Composite artefacts (combinations of primitives)
- [ ] Animation systems
- [ ] Audio-reactive artefacts
- [ ] Community submissions

## Phase 6: Real-World Deployments (Ongoing)
**Goal:** Prove practical value

- [ ] IoT sensor dashboard
- [ ] Business intelligence VR
- [ ] Scientific visualization
- [ ] Education/training tools

---

*Current Phase: 0 — Foundation*
