# Bar Chart Example

A 3D bar chart showing monthly revenue data.

## Data

8 months of revenue data, bar height mapped to value.

## Features

- Column bars arranged in a grid
- Color mapped to value (viridis scale)
- Hover glow effect
- Platform base with grid lines
- Value markers ($0, $50K, $100K)

## Running

Open `index.html` in a browser or serve via:

```bash
npx serve
```

Navigate to `/bar-chart/`
