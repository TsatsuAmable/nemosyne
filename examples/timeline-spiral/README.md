# Timeline Spiral Example

A 24-hour activity pattern visualized as a rising spiral.

## Concept

Time flows upward in a spiral pattern:
- **Spiral radius** — Time of day (tighter as day progresses)
- **Height (Y)** — Temporal progression
- **Color** — Activity level (red = inactive, teal = active)
- **Size** — Intensity of activity

## Data

24 hourly data points showing daily activity patterns.

## Features

- Rising spiral layout
- Diverging color scale (red for low, teal for high)
- Connecting lines between hours
- Central rotating core

## Running

Open `index.html` in a browser or serve via:

```bash
npx serve
```

Navigate to `/timeline-spiral/`
