## What is Nemosyne?

Nemosyne is a Web-Native VR framework for creating interactive data visualizations in three dimensions. Built on A-Frame, it transforms complex datasets into spatial experiences.

### Key Features
- **Artefact-Based:** Represent data as interactive 3D objects
- **Declarative:** Define scenes with simple JSON
- **Real-Time:** WebSocket and live data support
- **VR-First:** Spatial computing as default context
- **Cross-Platform:** Desktop, mobile, and VR headsets

## Core Concepts

### Artefacts
The fundamental unit of visualization. An artefact represents a data entity in 3D space.

### The Datumplane
A virtual dataspace for information manipulation, inspired by the Hyperion Cantos.

### Layouts
Algorithms for positioning data in 3D space (grid, radial, force-directed, etc.).

### Transforms
Data-to-visual mappings (scale, colour, position based on values).

### Behaviours
Interaction patterns (hover, click, animations).
